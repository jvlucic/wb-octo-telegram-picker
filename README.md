# Unidesq

## Instalation

* Install node 4+
* git clone project
* npm install

## Run

* Development: npm start
* Production: npm run build && npm run start:prod

## Tests and Linters

* All tests:    	npm test
* Watch tests: 		npm run test:watch
* All Linters: 		npm run lint
* Linter JS: 			npm run lint:js
* Linter Css: 		npm run lint:css
